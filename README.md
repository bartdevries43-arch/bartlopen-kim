# bartlopen — Run Coach (Kim)

Persoonlijk halve-marathontrainingsschema ("Op naar 21,1K") met coaching, stats en
badges. Van 0 naar de Halve Maassluis (17 januari 2027) in 28 weken, 2 trainingen
per week (woensdag + zaterdag). Statische web-app — werkt op telefoon, laptop en
iPad, en is te "installeren" als app.
